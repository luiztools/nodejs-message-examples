# nodejs-twilio
Simple Twilio API examples in Node.js.

## SMS
Tutorial: https://www.luiztools.com.br/post/como-enviar-sms-em-node-js-via-twilio/

To run:
- cd sms
- create the .env file with your settings
- change the destination number in the index.js
- npm i
- npm start
- SMS sent!
